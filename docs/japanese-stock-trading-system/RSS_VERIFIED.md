#### B-1. 関数方式の場合

```
関数名:       (例) RssOrder
セル例:       (例) =RssOrder("7203", "BUY", 100, 2540, "LIMIT", "当日")
引数の順序:
  1. (例) 銘柄コード           型:文字列  例:"7203"
  2. (例) 売買区分             型:文字列  例:"BUY" / "SELL" / 1 / 2
  3. (例) 数量                 型:整数    例:100
  4. (例) 価格                 型:数値    例:2540
  5. (例) 執行条件             型:文字列  例:"LIMIT" / "MARKET"
  6. (例) 有効期限             型:文字列  例:"当日" / "今週"
  7. (例) 口座区分             型:文字列  例:"特定" / "NISA" / "一般"
  8. ...
戻り値:
  - 成功時:  (例) 注文番号の文字列 "20260415-0001"
  - 失敗時:  (例) "#ERR!" セル値 / 負の数値 / エラーコード文字列
```

#### B-2. 発注シート方式の場合

```
シート名:     (例) "注文入力"
入力セル:
  - B2: 銘柄コード
  - B3: 売買区分
  - B4: 数量
  - B5: 価格
  - B6: 執行条件
  - ...
トリガー:     (例) B10 に "送信" と入力 / ボタン "発注" をクリック
結果セル:
  - D2: 状態 ("受付" / "エラー")
  - D3: 注文番号
  - D4: エラー内容
```
